# Customer Behavior Analysis using SQL, Python & Power BI

## Overview
This project focuses on analyzing customer behavior using Python, SQL, and Power BI. The workflow includes data loading, data cleaning, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard creation to generate meaningful business insights and support data-driven decision-making.

---

This dataset contains customer purchasing behavior, sales trends, customer segmentation, and transaction-related information used for analysis and visualization.

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (PostgreSQL / MySQL / SQL Server)
- Power BI
- Jupyter Notebook
- Gamma AI (Presentation Creation)

---

## Project Steps
1. Loaded and explored the dataset using Python.
2. Performed data cleaning and preprocessing.
3. Conducted Exploratory Data Analysis (EDA).
4. Imported data into SQL database and executed analytical SQL queries.
5. Built an interactive Power BI dashboard for visualization.
6. Created a business insights report and presentation using Gamma AI.

---

## Dashboard
The Power BI dashboard provides:
- Customer behavior insights
- Sales and revenue analysis
- Customer segmentation
- KPI tracking and trend analysis
- Interactive filters and visualizations

---

## Results
The project helped identify customer purchasing patterns, sales trends, and key business insights that can support better marketing and business strategies.

---

## How to Run
1. Download the dataset and project files.
2. Run the Python notebook for data cleaning and EDA.
3. Import the dataset into PostgreSQL/MySQL/SQL Server.
4. Execute the SQL queries provided in the project.
5. Open the Power BI `.pbix` file to view the dashboard.
